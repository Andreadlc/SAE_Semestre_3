## 3. Contraintes

### Technologiques :
- L'application doit être développée en PHP et utiliser une base de données MySQL ou un autre système SGBD compatible.
- Elle sera déployée sur un Raspberry Pi 4. Nous devrons préparer la carte SD avec :
    - Le système d'exploitation.
    - Un serveur web (ex : Apache).
    - Un serveur SGBD (ex : MySQL).
    - Une configuration sécurisée des accès SSH.

### Sécurité :
- Accès SSH sécurisé.
- Login et mot de passe obligatoires pour les administrateurs.
- Aucune modification des identifiants par défaut ne sera tolérée pour l’administrateur système.

### Développement et Déploiement :
- Chaque groupe doit mettre à disposition les livrables sur un dépôt GitHub ou GitLab, incluant la documentation, le code et les instructions d'installation.
- Il est interdit de tenter de hacker le matériel d’un autre groupe. Toute violation sera lourdement sanctionnée.

### Respect des délais :
- Les travaux doivent être remis dans les délais fixés par les enseignants. Les retards seront sévèrement pénalisés.
